## Project Overview

This portal helps students and admins manage day-to-day campus issues quickly:
- Submit and track complaints
- Report lost or found items
- Manage requests through an admin panel

The project is built with a simple and lightweight frontend stack for easy demonstration and deployment.

## Tech Stack

- HTML5
- CSS3
- JavaScript (Vanilla JS)
- LocalStorage (for temporary data handling)

## Main Modules

1. **Home (`Index.html`)**
   - Landing page with quick navigation to all modules.

2. **Lost & Found (`lost-found.html`)**
   - Add and view lost/found item details.

3. **Complaints (`complaints.html`)**
   - Raise student complaints and monitor status.

4. **Admin Panel (`admin.html`)**
   - Review and manage submitted data.

5. **Login (`login.html`)**
   - Basic access flow for user/admin side navigation.

6. **About (`about.html`)**
   - Project mission, modules, and contributors.

## Project Files

- `Index.html` – main landing page
- `about.html` – project description and team info
- `admin.html` – admin dashboard
- `complaints.html` – complaint handling module
- `login.html` – login module
- `lost-found.html` – lost and found module
- `script.js` – shared JavaScript logic/utilities
- `style.css` – shared styling

## How to Run

1. Download or clone the project.
2. Open the project folder.
3. Run `Index.html` directly in your browser.

> No extra installation is required.
